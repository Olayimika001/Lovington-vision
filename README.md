# Lovington Vision — Yoruba Farm Store (Vite + Vue + Tailwind)

**Lovington Vision Int. Ltd** — Yoruba Language Consulting & Sustainable Agriculture.  
Built with **Vite**, **Vue 3**, and **Tailwind CSS**.

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` – start dev server
- `npm run build` – build for production
- `npm run preview` – preview production build

## Stack

- Vue 3 (Composition API)
- Vue Router
- Pinia (with persisted state for cart)
- Vue I18n (English + Yorùbá)
- Tailwind CSS + custom CSS variables
